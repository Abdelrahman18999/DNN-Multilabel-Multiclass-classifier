# DNN-Multilabel-Multiclass-classifier
Using "Functional API method" to build an architecture of multi-label multi-class classifier

## Input features are:
- Height
- Weight
- Nose surface area

## Output:
The O/P will be two classes :
- Age category(baby, toddler, preteen, etc...)
- Gender (male, female)
